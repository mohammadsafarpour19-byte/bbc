# Beauty Smart Suite — Demo
دموی **فرانت‌اند** شامل ۶ قابلیت نوآورانه اسکین‌کر:
- Skin‑Match AI (پرسشنامه → پیشنهاد)
- Routine Builder (روتین صبح/شب)
- اشتراک ماهانه (پک جعبه‌ای)
- مشاور زنده (چت‌بات ساده + دکمهٔ اتصال)
- اسکن QR/بارکد (ZXing) با دمو INCI/EAN
- Virtual Try‑On (آپلود سلفی → قبل/بعد با فیلتر روشنایی/نرمی)

## استقرار
- GitHub Pages / Vercel / Netlify (استاتیک). فایل‌ها را در ریشه آپلود کنید.

## نکات
- برای اسکن QR روی موبایل، مجوز دوربین لازم است.
- برای تست QR، یک QR با متن `INCI: niacinamide, hyaluronic acid` بسازید.
- محصولات در `products.json` قابل تغییر هستند.
